VPN CLARO - INVITADOS

Instructivo para Usuario Contratista

El presente documento lo asistirá al momento de realizar el enrolamiento en Microsoft Authenticator desde la App móvil y la configuración para el cliente vpn.

1 DESCARGA DEL CLIENTE FORTICLIENT

El primer paso es tener instalado el cliente vpn FortiClient que puede descargar del siguiente link si es necesario:

Descargar FortiClient
Nota: En el caso de no tener permisos que se requiere para la instalación deberá solicitarle a su área de soporte IT la asistencia necesaria.
2 CONFIGURACIÓN FORTICLIENT

Se deberá configurar con los siguientes parámetros FortiClient VPN:

  • Remote Gateway: 172.28.37.249
  • Connection Name: VPN Claro Proveedores
  • Habilitar: Single Sign On (SSO) para VPN

Captura de Pantalla: Configuración de VPN
3 REINICIAR CLIENTE VPN

Para que los cambios se apliquen correctamente y se visualice la opción de SAML Login, se debe reiniciar el cliente FortiClient. Verifique que aparezca el botón correspondiente en la interfaz.


Captura de Pantalla: Botón SAML Login
4 INVITACIÓN Y DOBLE FACTOR (2F)

Para acceder a las aplicaciones y recursos de AMX Argentina S.A., use el código de verificación enviado a su correo. El código solamente funcionará durante 30 minutos.

ACLARACIÓN IMPORTANTE: En algunos casos se genera una "doble" autenticación: le envía un correo con un código de verificación similar al de la imagen.

Ejemplo de correo: Código de verificación de cuenta
5 CONEXIÓN FINALIZADA

Finalizada la carga de FortiClient veremos que en el navegador se nos abrirá una nueva ventana indicando que está conectado a la VPN. Ya podrá navegar en las distintas URLs y herramientas de la compañía.
