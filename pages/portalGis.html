PORTALGIS - CLARO

Instructivo de uso y visualización de coberturas

Este documento detalla el procedimiento para ingresar a la plataforma PortalGis y configurar correctamente las capas de cobertura para el análisis de red.

1 ACCESO A LA PLATAFORMA

Para ingresar a la plataforma debe realizarse por el siguiente Link:

IR A PORTALGIS

Para logearse debe hacerse con el usuario de red y la contraseña correspondiente[cite: 3].

Imagen: Interfaz de Login PortalGis
2 SELECCIÓN DE MÓDULO

Una vez dentro del sistema, se debe seleccionar el modulo “BANDEJA DE ELEMENTOS”[cite: 4].

Imagen: Panel de Selección de Módulos
3 BÚSQUEDA DE DIRECCIÓN

En el siguiente paso se visualiza las opciones de las cobertura y la búsqueda para colocar la dirección a revisar[cite: 5].

Imagen: Barra de búsqueda y visualización de mapa
4 CONFIGURACIÓN DE CAPAS

En las COBERTURAS TEMATIZADAS se deben tildar las siguientes casillas[cite: 6]:

  • • FRAUDES / ZONA_PELIGROSA / EDIFICIO_DISPONIBLE
  • • GPON / GPON_PLANIFICADO / GPON_SATURADO / ZONA_DESHABILITADA

En el item desplegable de SPLITTERS se deben seleccionar todas las casillas[cite: 8].

Imagen: Configuración de capas y filtros laterales
5 ANEXO: POLÍGONOS DE CAPACIDAD

Los polígonos determinan la capacidad y disponibilidad de la cuadra, así como también delimitan desde donde hasta donde puede tomarse el servicio de la misma[cite: 9, 10].

IMPORTANTE: Si el polígono se encuentra delineado con rojo se refiere a que el polígono no es ampliable (No se puede escalar para agregar otra caja en la cuadra).
Imagen: Ejemplo de polígono rojo no ampliable
